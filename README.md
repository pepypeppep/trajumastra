## Migrate Database
``` php artisan migrate:fresh ```

## Seeding Datasets (From CSV)
``` php artisan seed:csv "PATH DIRECTORY" ```

Example
Your datasets saved in project-directory/laravel-project/storage/app/public/trajumastra_db, then you can run
``` php artisan seed:csv  D:\Development\laragon\www\trajumastra-remake\storage\app\public\trajumastra_db ```
